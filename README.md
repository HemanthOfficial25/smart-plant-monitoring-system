# Smart Plant Monitoring System using ESP32

## Project Overview
This project monitors soil moisture using an ESP32 microcontroller and automatically activates a water pump when the soil becomes dry.

## Features
- Real-time soil moisture monitoring
- Automatic irrigation
- ESP32-based control
- Low-cost implementation
- Suitable for smart agriculture applications

## Components Used
- ESP32
- Soil Moisture Sensor
- Relay Module
- Water Pump
- Breadboard
- Jumper Wires

## Working Principle
The soil moisture sensor continuously measures moisture levels in the soil. When the soil becomes dry, the ESP32 activates the relay, which turns on the water pump. Once adequate moisture is detected, the pump is automatically switched off.
